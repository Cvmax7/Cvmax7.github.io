2. 证明除法算法

2.1 除法算法

**对任意给定的整数a和b，其中b > 0，存在唯一的整数对q（商）和r（余数）使得， **
$$
a = qb + r
$$
且 0 ≤ r < b。

2.2 证明过程

2.2.1 证明存在性

构造集合
$$
S = {a-bk:k\in \Z\ 且\ a-bk \ge 0 }
$$
显然，集合S非空，由良序原则，存在一个最小元$$r \in S\ 且 \ r=a-qb$$

因此，$$a = qb + r, r\ge 0$$。

我们采用反证法，假设最小元$$r>b$$，则

$$\exists t\in\Z\ s.t.$$
$$
r = tb + r^*
$$
则有
$$
a=qb+r=qb+tb+r^*=(q+t)b+r^*
$$
令 $$ q+t = q^* \in Z$$，则有
$$
a=q^b+r^
$$
此时存在$$r^*<r$$,则此时的最小元为$$r^*$$,与假设矛盾。

综上: $$0\leq r< b$$

Q.E.D

2.2.2 证明唯一性

我们才用反证法，假设存在两对$$(q_1,r_1)$$,$$(q_2,r_2)$$,满足$$a=qb+r$$,其中$$q_1\ne q_2,\ r_1\ne r_2,\ 0\leq r_1 < b,\ 0\leq r_2 < b$$，则有：
$$
q_1b+r_1=q_2b+r_2
$$
处理得：
$$
(q_1-q_2)b=(r_2-r_1)
$$
即
$$
b\mid(r_2-r_1)\
$$
又
$$
\because 0\leq r_1 < b,\ 0\leq r_2 < b\
\therefore -b<r_2-r_1<b\
\therefore r_2-r_1 = 0
$$
与假设不符，故只存在一对$$(q,r)\ s.t.\ 0\leq r < b\ 且\ a=qb+r$$

Q.E.D

3.1 迭代版本的gcd

#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>

int gcd(int a, int b);

int main()
{
    int a, b;
    printf("请输入a,b\n");
    scanf("%d,%d",&a,&b);
    printf("a和b最大公因数为：%d", gcd(a, b));
    return 0;
}

int gcd(int a, int b)
{
    int temp;
    if (a < b)
    {
        temp = b;
        b = a;
        a = temp;
    }
    while (a % b)
    {
        temp = b;
        b = a % b;
        a = temp;
    }
    return b;
}

3.2 迭代版本的egcd

#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>

// ra+sb = gcd(a,b)
void egcd(int a, int b);

int main()
{
    int a, b;
    printf("请输入a,b\n");
    scanf("%d,%d", &a, &b);
    egcd(a, b);
    return 0;
}

void egcd(int a, int b)
{
    int s1 = 1;
    int r0 = 1;
    int s0 = 0;
    int r1 = 0;
    int q;
    int temp;
    int res_r;
    int res_s;
    int res_gcd;
    while (b)
    {
        q = a / b;
        temp = a;
        a = b;
        b = temp % b;
        temp = r0;
        r0 = r1;
        r1 = temp - q * r1;
        temp = s0;
        s0 = s1;
        s1 = temp - q * s1;
    }
    res_r = r0;
    res_s = s0;
    res_gcd = a;
    printf("r=%d,s=%d,gcd(a,b)=%d", res_r, res_s, res_gcd);
}

4. 第二章的第6、8题

4.1 第6题

  • 假设$$g^a\equiv1\ (mod\ m)\ 且\ g^b\equiv 1(mod\ m)$$,请证明：$$g^{gcd(a,b)}\equiv 1(mod\ m)$$。

证明：

由$$B\acute{e}zout$$定理可得：
$$
gcd(a,b)=ar+bs
$$
且a与b的最大公因子是唯一的。

即证：
$$
g^{ar＋bs}\equiv1(mod\ m)\
$$
又
$$
g^a\equiv1\ (mod\ m)\ \Rightarrow\ g^a=k_1m+1\
g^b\equiv1\ (mod\ m)\ \Rightarrow\ g^b=k_2m+1
$$
又：
$$
g^a·g^b = (k_1m+1)·(k_2m+1)=k_1k_2m+k_1m+k_2m+1\
(g^a·g^b)\ mod\ m =(k_1k_2m+k_1m+k_2m+1)\ mod\ m =1
$$
故
$$
g^{a+b}\equiv1(mod\ m)
$$
即证：
$$
(g^a)^r\equiv 1(mod\ m)\
(g^b)^s\equiv 1(mod\ m)
$$

$$
\because(g^a)^r=(k_1m+1)^r=C_r^0(k_1m)^r1^0+C_r^1(k_1m)^{r-1}1^1+…+C_r^{r-1}(k_1m)^11^{r-1}+C_r^r(k_1m)^01^r\
\ \ \ \ \ (g^b)^s=(k_2m+1)^s=C_s^0(k_2m)^s1^0+C_s^1(k_2m)^{s-1}1^1+…+C_s^{s-1}(k_2m)^11^{s-1}+C_s^s(k_2m)^01^s\
$$

显然，
$$
[C_r^0(k_1m)^r1^0+C_r^1(k_1m)^{r-1}1^1+…+C_r^{r-1}(k_1m)^11^{r-1}+C_r^r(k_1m)^01^r] \ mod\ m =\ C_r^r(k_1m)^01^r\ mod\ m=1
$$

$$
\therefore (g^a)^r\equiv 1(mod\ m)\
$$

同理可得：
$$
(g^b)^s\equiv 1(mod\ m)
$$

$$
\therefore (g^a)^r·(g^b)^s \equiv 1(mod\ m) \Leftrightarrow g^{ar＋bs}\equiv1(mod\ m)\Leftrightarrow g^{gcd(a,b)}\equiv1(mod\ m)
$$

Q.E.D

4.2 第8题

  • 证明：如果$$gcd(a,b)=d$$,则$$gcd(a/d,b/d)=1$$

证明：

因为$$gcd(a,b)=d$$，可知a与b的最大公因数为d。

Ⅰ.当$$a\neq b$$时

设 $$\exists t_1,t_2 \in Z\ 且 t_1\neq t_2 \neq 0\ ,\ s.t.$$
$$
a=t_1d\
b=t_2d\
$$
$$\because$$ 假设 $$gcd(t_1,t_2)=k\neq 1$$,则必$$\exists d^* = kd,s.t.\ d^*>d$$，与题意不符，故$$gcd(t_1,t_2)=1$$。

即$$t_1$$与$$t_2$$互素。

又：
$$
a/d = t_1\
b/d = t_2
$$
故 $$gcd(a/d,b/d)=1$$。

Ⅱ.当$$a=b$$时

易知：$$a=b=d$$

则 $$gcd(a/d,b/d)= gcd(1,1)=1$$

Q.E.D
